Dr. Layer 1.0
Exercise 2- Loading

Introduction

Introduction

Background

An understanding of the various loading types both ideal and practical is necessary for understanding the effects that will manifest in wave propagation mechanics of the travel media.

 BACK TO TOP

Objective

This exercise will show how Dr. Layer attempts to visually display the effects of varying load types. The objective is to appreciate the wave propagation characteristics that can be generated when using different loading conditions and when changing the parameters that specify each loading condition.

 BACK TO TOP

Things to Do

  1. Open the Dr. Layer program. By default we get twelve layers. The top six layers are hardwired into the system with a very fast velocity. The bottom six layers are hardwired with a very slow velocity. The input loading is such that we can see its motion on the plot attached to the base node on the lowest layer, Figure 1.


    Figure 1
  2. For the purpose of this exercise we can use a soil layer with a single wave speed. To get this soil condition, use the select all option in the edit menu. Next, go to the wave speed menu option and choose say, very slow.

  3. There are four different loading types hardwired into the program. Default option comes up as sinusoidal. Observe the type of wave motion obtained.

  4. Try changing the loading type to a ramp or a pulse type loading. Note what you observe. To help identifying the wave propagation characteristics due to different loading conditions use additional displacement plots at different heights as shown in Figure 2.


    Figure 2

  5. The option for an earthquake load has a hardwired real earthquake loading type input. You are however encouraged to try obtaining an earthquake record (e.g. from the internet) save it as a earthquake.txt file. You can upload this file and note what you observe.

  6. Change the default values for the amplitude and frequency. What do you observe?

  7. Using only the pulse loading go to the options menu and make the base unreleased. Do you notice any difference in the input loading plots?

  8. Repeat the process above for the ramp load.

 BACK TO TOP

Observation

  1. Different loading conditions generate very different loading patterns. Each one help interprete the phenomenon of wave propagation in layered media.

  2. In Dr. Layer the earthquake load is input as a force. The plot attached to the bottom layer represents the resulting displacement at the bottom of the soil profile.
